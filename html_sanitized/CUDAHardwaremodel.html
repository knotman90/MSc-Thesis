CUDA Hardware model

There are various kind of CUDA capable device on the market. From a mid-level laptop graphic card to a numerical computation dedicated card.17 The Nvidia GPU hardware architecture is built around a scalable array of multithreaded Streaming Multiprocessors (SMs). The Streaming Multiprocessors is designed to execute hundreds of threads concurrently and contains a number of Streaming Processors (SP)18.
