Texture Memory

As its name suggests, texture memory is used for storing textures. Although this memory was designed for the classical openGL and DirectX graphic pipeline (see section [graphicPipeline]) it can be used successfully in some scenario to accelerate the application especially when the memory access pattern exhibit 2D-spatial 31 locality it will give higher memory bandwidth by reducing requests to the off-chip DRAM. Threads of the same warp that read texture addresses that are close together, will achieve better performance using the texture cache [@NvidiaprogGuide].
