Nvidia Visual Profiler

The NVIDIA Visual Profiler is a cross-platform performance profiling tool that delivers developers feedback for optimizing CUDA C/C++ applications, identify potential performance bottleneck issues and plus it provides optimization suggestions. These informations are organized in tables that show activity occurring on both CPU and GPU in a unified time line, including CUDA API calls, memory transfers and CUDA kernel launches and each information is coupled with many other statistics and metrics, like timing or occupancy. This tools has access to the low level metric collected directly from the hardware and low-level software instrumentation, like power, thermal or clock situation of the devices.
