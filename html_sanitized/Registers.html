Registers

r 0.45

The registers on the GPU are general purpose. Each SM has a number of registers to share between its cores. If too much register space is used by a kernel, the number of cores per SM that can be utilized is lowered or local memory 36 can be utilized. The Geforce GTX 480 has 32.768, 32-bit registers on each of the 15 SMs37 and that memory is 8x faster38 even than the already very fast shared memory.
