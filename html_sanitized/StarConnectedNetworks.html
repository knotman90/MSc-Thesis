Star-Connected Networks

Star topology (see figure [fig:starMesh]) every node is connected to a central node that acts as the central processor (as server if we consider the analogy with the LAN) . It is similar to the bus-based (see section [bus-based]) network, because communication between any pair of nodes is routed through the central processor (and the latter is the shared medium the all the other nodes share) and because the central processor is the bottleneck in this topology and also the single point of failure (if this node goes down the entire network stops to work).
