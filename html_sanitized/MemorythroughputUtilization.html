Memory throughput Utilization

Memory transfer should be avoided, but some of them are unavoidable especially if they are performed in low bandwidth. For instance host to/from device bandwidth is much lower than device to device one. One way to minimize data transfer between the host and the device is to move more code from the host to the device, even if that means running kernels with low parallelism computations and try to batch many small transfers into a single large one, that always perform better.
